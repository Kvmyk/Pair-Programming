# Dev Container Configuration

## Zainstalowane rozszerzenia

1. **ms-python.python** - Podstawowe rozszerzenie dla Pythona
2. **ms-vsliveshare.vsliveshare** - KLUCZOWE: Automatyczna instalacja Live Share do współpracy
3. **EditorConfig.EditorConfig** - Utrzymanie spójnego stylu kodu
4. **tomoki1207.pdf** - Wyświetlanie plików PDF bezpośrednio w VS Code

## Informacje

- **Obraz**: `mcr.microsoft.com/devcontainers/universal:latest`
- **Użytkownik**: `vscode`
- **Środowisko**: Python 3.11

## Użycie

Po uruchomieniu kontenera użyj Live Share (ikona ludzików) do rozpoczęcia sesji współpracy.
